# Credit Risk Modelling

## Overview
This project builds a credit risk model using loan-level data and logistic regression. It includes data preparation, feature engineering, model training, and evaluation, designed for banking and risk analytics use cases.

## Features
Source data → ETL → Staging → Data Warehouse → Data Marts → Features → Models → Validation → Reporting
1. Data Collection
2. Data Cleaning / Data Wrangling
3. Data Integration / Harmonisation
4. Data Transformation (DPD, delinquency, exposures)
5. Feature Engineering (behavioural variables)
6. Target Variable Construction (PD, LGD, EAD)
7. Dataset Splitting (train/test/validation)
8. Model Development (PD/LGD/EAD)
9. Model Evaluation & Validation
10. Calibration
11. Documentation & Data Lineage
12. Implementation (SQL/Python pipelines)
13. Monitoring & Backtesting
